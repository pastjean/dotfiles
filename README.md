# dotfiles

Your preferences should be kept in your pocket like you care for them.

## Nice stuff to backup


- OpenVPN profiles
- SSH keys
- PGP keys
- ...

## Install

```sh
git clone git@github.com:pastjean/dotfiles.git "$HOME/.dotfiles"
cd $HOME/.dotfiles
bash install.sh
```
